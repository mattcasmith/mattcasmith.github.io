This publisher offers a free IT and programming ebook every day

2018-08-18  Cyber Security,   Programming

My blog has had a bit of a cyber security learning theme recently (I have a couple more posts lines up on the topic, too) and it’s only set to continue this week with a great free resource that I first learnt about at the SANS Cyber Retraining Academy.

I’d never heard of it before, but Packt Publishing is a UK-based publisher specialising in IT that has released more than 6,000 books to help industry professionals learn about new technologies and boost their knowledge of the ones they’re already using.

The best bit – and the reason I’m writing this post – is that they offer a free ebook on their website every single day. These expire daily, so you need to remember to check, and they’re not always strictly about cyber security, but you can’t argue with a free resource like this.

I’ve been quite lazy about checking their offerings since I learnt about the site more than a year ago, but since I returned a month or two ago I’ve picked up books including:

  • Advanced Machine Learning with Python
  • Building Machine Learning Systems with Python
  • What You Need to Know About Machine Learning
  • Hadoop Explained
  • Practical Data Analysis
  • Learning Python
  • Mastering Python
  • Python Data Structures and Algorithms
  • Python High Performance
  • Raspberry Pi Cookbook for Python Programmers
  • What You Need to Know About Python
  • Creating Your MySQL Database
  • Practical Web Development
  • Thinking in CSS
  • Thinking in HTML
  • Thinking in JavaScript

As you can see, there’s quite a variety and usually the focus is more on coding than security, but they’re usually pretty useful, and if you travel for work as much as I do then they’re great to have on your smartphone to make use of all of those train journeys.

On top of the daily free book, many of the titles listed above are also permanently free at the bottom of the download page, so if you’re stuck for some technical reading then there’s bound to be something of interest available for you to pick up there.

It’s also worth keeping an eye out at Humble Bundle, which recently offered a bundle of more than ten Wiley cyber security ebooks for $15 (about £12). I’m currently reading Threat Modelling: Designing for Security, which I’ll post a review of when I’m done, and next on my list is The Art of Memory Forensics, which comes highly recommended from one of my colleagues.
